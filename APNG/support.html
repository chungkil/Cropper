사용 방법

  1. 앱을 실행하고 사진 접근 권한을 허용합니다.
  2. Live Photo 선택 버튼을 탭하고 Live Photo를 한 장 고릅니다.
  3. 자동으로 프레임이 추출되고 APNG로 변환됩니다.
  4. 변환된 APNG는 공유 시트로 보내거나, 메인 화면의 내 파일에서 다시 열어 확인할 수 있습니다.
  5. 파일 앱 → "이 iPhone 내" → APNG 폴더에서도 직접 접근할 수 있습니다.

자주 묻는 질문 (FAQ)

Live Photo를 선택했는데 변환되지 않습니다.

선택한 사진이 실제로 Live Photo인지 확인해 주세요. iCloud에서 원본을 내려받지 못한 경우에도 실패할 수 있으니, Wi-Fi 환경에서 다시 시도하시거나 사진 앱에서 해당 사진을 한 번 열어 원본을 캐시한 뒤 다시 시도해 보세요.

변환된 APNG 파일은 어디에 저장되나요?

앱의 Documents 폴더에 저장되며, iOS의 파일 앱에서 이 iPhone 내 → APNG 경로로 직접 접근할 수 있습니다. 앱 내 내 파일 화면에서도 동일한 목록을 확인하실 수 있습니다.

출력 파일의 해상도와 프레임레이트는 어떻게 되나요?

현재 기본 설정은 FPS 24, 가로 360px 다운샘플입니다. 용량과 호환성을 위한 균형 잡힌 값이며, 원본 해상도는 변환 후 정보 화면에서 확인할 수 있습니다.

광고를 제거할 수 있나요?

현재 버전은 무료 광고 지원 모델로 제공됩니다. 향후 업데이트에서 광고 제거 옵션이 추가될 수 있습니다.

App Tracking Transparency(ATT) 권한은 왜 묻나요?

iOS의 광고 식별자(IDFA)를 사용해 더 관련성 있는 광고를 보여주기 위해 필요합니다. 거부하셔도 앱의 모든 기능은 동일하게 동작합니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.

사진 접근 권한을 거부했어요. 다시 허용하려면?

iOS 설정 → APNG → 사진에서 권한을 변경할 수 있습니다. "선택된 사진" 또는 "전체 사진 접근"으로 설정해 주세요.

변환된 파일을 삭제하려면?

앱 내 내 파일 화면에서 해당 항목을 스와이프하거나, 파일 앱에서 직접 삭제할 수 있습니다.

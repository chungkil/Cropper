라이브포토 변환기 Support

도움이 필요하신가요?

사용 방법

  1. 앱을 실행하고 사진 접근 권한을 허용합니다.
  2. 라이브 포토 선택하기 버튼을 탭하고 Live Photo를 한 장 고릅니다.
  3. APNG · GIF · WebP · 동영상(mp4) 중 원하는 출력 포맷을 선택합니다.
  4. 변환된 파일은 공유 시트로 보내거나, 메인 화면의 내 파일에서 다시 열어 확인할 수 있습니다.
  5. 파일 앱 → "이 iPhone 내" → 라이브포토 변환기 폴더에서도 직접 접근할 수 있습니다.

자주 묻는 질문 (FAQ)

Live Photo를 선택했는데 변환되지 않습니다.

선택한 사진이 실제로 Live Photo인지 확인해 주세요. iCloud에서 원본을 내려받지 못한 경우에도 실패할 수 있으니, Wi-Fi 환경에서 다시 시도하시거나 사진 앱에서 해당 사진을 한 번 열어 원본을 캐시한 뒤 다시 시도해 보세요.

변환된 파일은 어디에 저장되나요?

앱의 Documents 폴더에 저장되며, iOS의 파일 앱에서 이 iPhone 내 → 라이브포토 변환기 경로로 직접 접근할 수 있습니다. 앱 내 내 파일 화면에서도 동일한 목록을 확인하실 수 있습니다.

어떤 포맷으로 변환할 수 있나요?

APNG(고화질·투명), GIF(범용), WebP(고효율) 이미지와 동영상(mp4)을 지원합니다. APNG·GIF·WebP는 소리가 없는 이미지 포맷이며, 소리까지 담으려면 동영상(mp4)을 선택하세요. 이미지 포맷은 기본 FPS 24, 가로 360px로 다운샘플됩니다.

변환한 APNG가 사진 앱에서 안 움직여요.

iOS 사진 앱은 APNG의 애니메이션을 정적 이미지로만 표시합니다(시스템 제한). 앱 내 미리보기나 Safari, macOS 미리보기 등에서 애니메이션이 정상 재생됩니다. 어디서나 움직이길 원하시면 GIF나 동영상(mp4)을 사용하세요.

광고를 제거할 수 있나요?

현재 버전은 무료 광고 지원 모델로 제공됩니다. 향후 업데이트에서 광고 제거 옵션이 추가될 수 있습니다.

App Tracking Transparency(ATT) 권한은 왜 묻나요?

iOS의 광고 식별자(IDFA)를 사용해 더 관련성 있는 광고를 보여주기 위해 필요합니다. 거부하셔도 앱의 모든 기능은 동일하게 동작합니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.

사진 접근 권한을 거부했어요. 다시 허용하려면?

iOS 설정 → 라이브포토 변환기 → 사진에서 권한을 변경할 수 있습니다. "선택된 사진" 또는 "전체 사진 접근"으로 설정해 주세요.

변환된 파일을 삭제하려면?

앱 내 내 파일 화면에서 해당 항목을 스와이프하거나, 파일 앱에서 직접 삭제할 수 있습니다.

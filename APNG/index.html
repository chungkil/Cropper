라이브포토 변환기

Live Photo를 APNG · GIF · WebP · 동영상으로

App Store에서 다운로드

소개

라이브포토 변환기는 아이폰의 Live Photo를 APNG, GIF, WebP, 동영상(mp4)으로 변환하는 가장 간단한 방법입니다. 사진 라이브러리에서 Live Photo를 선택하고 원하는 포맷만 고르면 자동으로 변환합니다. 변환된 파일은 앱 안에서 바로 재생해보고, 다른 앱으로 공유하거나 파일 앱에 저장할 수 있습니다.

주요 기능

4가지 포맷 변환

APNG(고화질·투명), GIF(범용), WebP(고효율), 동영상(mp4, 소리 포함) 중 선택합니다.

소리까지 보존

동영상(mp4)으로 변환하면 Live Photo의 오디오를 그대로 담습니다.

앱 내 미리보기

변환 결과를 앱 안에서 바로 재생해 확인할 수 있습니다.

파일 관리

변환한 결과를 내 파일 목록에서 다시 보고 공유·삭제할 수 있습니다.

사용 방법

  1. 앱을 실행하고 라이브 포토 선택하기를 탭합니다.
  2. 사진 라이브러리에서 변환할 Live Photo를 한 장 선택합니다.
  3. APNG · GIF · WebP · 동영상 중 원하는 포맷을 선택합니다.
  4. 변환이 끝나면 미리보기에서 확인하고 공유하거나 내 파일에서 다시 봅니다.

APNG

Live Photo를 움직이는 PNG로

소개

APNG는 아이폰의 Live Photo를 애니메이션 PNG(APNG)로 변환하는 가장 간단한 방법입니다. 사진 라이브러리에서 Live Photo를 선택하면 자동으로 프레임을 추출해 움직이는 PNG로 저장합니다. 변환된 파일은 파일 앱에서 확인하거나 다른 앱으로 공유할 수 있습니다.

주요 기능

Live Photo 변환

사진 라이브러리의 Live Photo를 APNG로 즉시 변환합니다.

고품질 출력

FPS 24, 가로 360px 다운샘플로 균형 잡힌 용량과 화질을 제공합니다.

파일 관리

변환된 APNG를 앱 내 목록에서 미리보기·재생·삭제할 수 있습니다.

공유 및 저장

공유 시트와 파일 앱(Documents) 노출로 자유롭게 활용하세요.

사용 방법

  1. 앱을 실행하고 Live Photo 선택을 탭합니다.
  2. 사진 라이브러리에서 변환할 Live Photo를 한 장 선택합니다.
  3. 자동으로 프레임을 추출해 APNG로 변환합니다.
  4. 완성된 APNG를 공유하거나 내 파일에서 다시 확인합니다.

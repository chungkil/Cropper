What is Onionote?

When you want to annotate an important PDF without overwriting the original, Onionote keeps the source untouched and records your notes on separate layers. Like Photoshop layers — toggle them off when you don't need them, or export only the ones you want.

Features

■

Layer System

The PDF stays untouched. Each layer records notes independently — toggle or export any subset.

◆

Apple Pencil-Optimized

Zero-lag Metal strokes. Pressure · tilt aware. Pencil Pro gestures supported.

☁

iCloud Sync

Same document state on every device. Manageable via the Files app.

⇪

Flexible Export

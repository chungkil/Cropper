❓ 자주 묻는 질문FAQよくある質問

Q. 화면이 자동으로 꺼지지 않나요?

Q. Does the screen stay on?

Q. 画面は自動でオフになりますか？

네, 앱 사용 중에는 화면이 자동으로 꺼지지 않습니다.

Yes, the screen stays on while using the app.

いいえ、アプリ使用中は画面が自動でオフになりません。

Q. 배터리 소모가 심한가요?

Q. Does it drain battery?

Q. バッテリー消費は激しいですか？

SwiftUI 모드를 사용하면 배터리 소모를 최소화할 수 있습니다. 장시간 사용 시에는 충전기 연결을 권장합니다.

Using SwiftUI mode minimizes battery consumption. For extended use, we recommend connecting to a charger.

SwiftUIモードを使用するとバッテリー消費を最小限に抑えられます。長時間使用する場合は充電器への接続をお勧めします。

Q. 잠자기 기능은 어떻게 작동하나요?

Q. How does the sleep timer work?

Q. スリープタイマーはどのように動作しますか？

설정한 시간이 지나면 자동으로 앱이 종료됩니다. 잠들기 전 사용하기 좋습니다.

The app automatically closes after the set time. Great for use before sleep.

設定した時間が経過すると自動的にアプリが終了します。就寝前の使用に最適です。

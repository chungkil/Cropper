  • 다양한 시각화 옵션 (바형, 원형, 타이머형)

자주 묻는 질문 (FAQ)

Q: D-Day는 몇 개까지 추가할 수 있나요?
A: 무제한으로 D-Day를 추가할 수 있습니다.
Q: 데이터는 어디에 저장되나요?
A: 모든 D-Day 데이터는 사용자의 iOS 기기에 안전하게 저장됩니다.
Q: 위젯을 추가하는 방법은 무엇인가요?
A: iOS에서는 홈 화면을 길게 누른 후 좌측 상단의 + 버튼을 탭하여 Day2U 위젯을 추가할 수 있습니다. 위젯의 크기와 표시할 D-Day를 선택할 수 있습니다.

플랫폼별 지원 기능

iOS

  • 홈 화면 위젯
  • 잠금 화면 위젯
  • 실시간 알림
  • 다양한 그래프 시각화
  • 직관적인 인터페이스

유용한 팁

배터리 최적화: 실시간 업데이트가 필요하지 않은 위젯은 수동 업데이트로 설정하여 배터리 사용량을 줄일 수 있습니다.
iCloud 동기화: 안정적인 동기화를 위해 iCloud 저장공간에 충분한 여유가 있는지 확인해주세요.

문제 해결

  1. 앱이 느리게 동작하는 경우:
    • 앱을 완전히 종료하고 다시 시작해보세요.
    • 기기를 재시작해보세요.
    • 최신 버전으로 업데이트되어 있는지 확인해주세요.
  2. 동기화가 안 되는 경우:
    • 기기의 iCloud 설정을 확인해주세요.
    • 인터넷 연결을 확인해주세요.
    • iCloud에 충분한 저장 공간이 있는지 확인해주세요.

Getting Started

Key Features

Frequently Asked Questions (FAQ)

Q: How many D-Days can I add?
A: You can add unlimited D-Days.
Q: Where is my data stored?
A: All D-Day data is securely stored on your iOS device.
Q: How do I add a widget?
A: On iOS, long-press the home screen and tap the + button in the top left corner to add a Day2U widget. You can select the widget size and which D-Day to display.

Platform Features

iOS

  • Home screen widgets
  • Lock screen widgets
  • Real-time notifications
  • Various graph visualizations
  • Intuitive interface

Useful Tips

Battery Optimization: Set widgets that don't need real-time updates to manual refresh to reduce battery usage.
iCloud Sync: Make sure you have enough iCloud storage space for reliable synchronization.

Troubleshooting

  1. If the app runs slowly:
    • Completely close and restart the app.
    • Restart your device.
    • Make sure you're updated to the latest version.
  2. If sync isn't working:
    • Check your device's iCloud settings.
    • Check your internet connection.
    • Make sure you have enough iCloud storage space.

よくある質問（FAQ）

Q: D-Dayはいくつまで追加できますか？

Q: データはどこに保存されますか？
A: すべてのD-DayデータはiOSデバイスに安全に保存されます。
Q: ウィジェットの追加方法は？
A: iOSではホーム画面を長押しして、左上の+ボタンをタップしてDay2Uウィジェットを追加できます。ウィジェットのサイズと表示するD-Dayを選択できます。

便利なヒント

バッテリー最適化： リアルタイム更新が不要なウィジェットは手動更新に設定してバッテリー消費を抑えることができます。
iCloud同期： 安定した同期のためにiCloudストレージに十分な空き容量があることを確認してください。

トラブルシューティング

  1. アプリの動作が遅い場合：
    • アプリを完全に終了して再起動してください。
    • デバイスを再起動してください。
    • 最新バージョンに更新されているか確認してください。
  2. 同期ができない場合：
    • デバイスのiCloud設定を確認してください。
    • インターネット接続を確認してください。
    • iCloudに十分なストレージ容量があるか確認してください。

수집하는 데이터

Heliosphere는 앱 품질 개선을 위해 다음의 데이터를 수집할 수 있습니다.

진단 데이터 익명

앱 성능 및 사용 패턴에 관한 익명화된 정보입니다.

  • 앱 실행 시간 및 성능 지표
  • 기기 모델 및 운영체제 버전
  • 앱 기능 사용 빈도

충돌 데이터 익명

앱 충돌 시 자동으로 수집되는 기술 정보입니다.

  • 충돌 로그 및 스택 트레이스
  • 충돌 시점의 기기 상태
  • 앱 버전 정보

데이터 수집 방법

이 데이터는 Apple의 표준 분석 프레임워크를 통해 수집됩니다. 사용자는 iOS/iPadOS 설정에서 분석 데이터 공유를 언제든지 비활성화할 수 있습니다.

설정 방법: 설정 → 개인정보 보호 및 보안 → 분석 및 개선 → "앱 분석 공유" 끄기

데이터 연결

수집되는 모든 데이터는 사용자의 신원과 연결되지 않습니다.

  • 개인 식별 정보를 수집하지 않습니다
  • Apple ID나 기기 고유 식별자와 연결하지 않습니다
  • 데이터 추적에 사용되지 않습니다

데이터 활용 목적

  • 앱 안정성 향상
  • 성능 최적화
  • 사용자 경험 개선
  • 버그 수정

Data We Collect

Heliosphere may collect the following data to improve app quality.

Diagnostics Anonymous

Anonymized information about app performance and usage patterns.

  • App launch time and performance metrics
  • Device model and OS version
  • Feature usage frequency

Crash Data Anonymous

Technical information automatically collected when the app crashes.

  • Crash logs and stack traces
  • Device state at time of crash
  • App version information

How Data Is Collected

This data is collected through Apple's standard analytics framework. Users can disable analytics data sharing at any time in iOS/iPadOS settings.

How to disable: Settings → Privacy & Security → Analytics & Improvements → Turn off "Share App Analytics"

Data Linkage

All collected data is not linked to your identity.

  • No personally identifiable information is collected
  • Not linked to Apple ID or device identifiers
  • Not used for tracking purposes

Purpose of Data Collection

  • Improve app stability
  • Optimize performance
  • Enhance user experience
  • Fix bugs

収集するデータ

Heliosphereは、アプリの品質向上のために以下のデータを収集する場合があります。

診断データ 匿名

アプリのパフォーマンスと使用パターンに関する匿名化された情報です。

  • アプリの起動時間とパフォーマンス指標
  • デバイスモデルとOSバージョン
  • 機能の使用頻度

クラッシュデータ 匿名

アプリがクラッシュした際に自動的に収集される技術情報です。

  • クラッシュログとスタックトレース
  • クラッシュ時のデバイス状態
  • アプリのバージョン情報

データ収集方法

このデータはAppleの標準分析フレームワークを通じて収集されます。ユーザーはいつでもiOS/iPadOSの設定で分析データの共有を無効にできます。

無効化の方法: 設定 → プライバシーとセキュリティ → 解析および改善 → 「Appデベロッパと共有」をオフ

データの関連付け

収集されるすべてのデータは、ユーザーのIDに関連付けられません。

  • 個人を特定できる情報は収集しません
  • Apple IDやデバイス識別子に関連付けません
  • トラッキング目的で使用しません

データ収集の目的

  • アプリの安定性向上
  • パフォーマンスの最適化
  • ユーザー体験の改善
  • バグの修正

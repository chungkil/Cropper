고시원 통합 관리 앱 사용 가이드

Google Play App Store

1 시작하기

  1. Google Play 또는 App Store에서 방주인을 설치합니다.
  2. Apple 계정(iOS) 또는 Google 계정(Android/iOS)으로 로그인합니다.
  3. 첫 로그인 시 고시원을 생성하거나, 기존 고시원의 초대 코드를 입력하여 참여합니다.
  4. 메인 화면 우측 하단 + 버튼을 눌러 첫 번째 시설을 등록하세요.
가입 즉시 60일간 프로 플랜(방 60개, 시설 무제한)을 무료로 체험할 수 있습니다.

2 시설 관리

  1. 시설 추가 — 메인 화면 + 버튼 → 시설 이름 입력 → 저장. 방을 개별 추가하거나 일괄 등록(시작~끝 번호)으로 한 번에 만들 수 있습니다.
  2. 시설 수정/삭제 — 시설 카드를 길게 누르면 수정 또는 삭제 메뉴가 나타납니다. 삭제 시 하위 방도 함께 삭제됩니다.
  3. 빈방 모아보기 — 메인 화면의 빈방 카드를 누르면 모든 시설의 빈방을 방 타입별로 한 눈에 확인할 수 있습니다.

3 방 관리

  1. 방 목록 — 시설 카드를 눌러 방 목록으로 이동합니다. 상단 필터(전체/사용중/빈방/예약/방타입)로 원하는 방만 볼 수 있습니다.
  2. 방 추가 — 방 목록 우측 하단 + 버튼을 누르면 새 방을 추가합니다.
  3. 방 상태 변경 — 방 상세에서 사용중 / 빈방 버튼으로 상태를 전환합니다. 사용중 → 빈방 전환 시 입실자 정보 초기화 여부를 선택할 수 있습니다.
  4. 방 찾기 — 기간과 옵션 조건을 설정하여 전체 시설에서 빈방을 검색할 수 있습니다.
  5. 입실자 검색 — 메인 화면의 검색 버튼으로 입실자 이름을 검색합니다.

4 입실자 & 계약 정보

  1. 기본 정보 — 방 상세에서 표준방세, 보증금, 입실일, 계약만료일을 입력합니다.
  2. 인원 정보 — 입실자 이름, 연락처, 입금자명을 기록합니다. 전화번호를 누르면 바로 전화를 걸 수 있습니다.
  3. 계약 캘린더 — 계약 기간이 월별 캘린더로 표시됩니다. 파란색은 사용중, 주황색은 예약 상태입니다.
  4. 예약 — 빈방 상태에서 예약 시작일과 종료일을 설정하면 해당 기간이 캘린더에 주황색으로 표시됩니다.

5 납부 관리

  1. 납부 상태 — 방 상세에서 완납 / 부분납 / 미납을 기록합니다. 수납 금액도 입력할 수 있습니다.
  2. 매출 현황 — 메인 화면에서 전체 매출, 미납/부분납/완납 현황을 바 그래프로 확인합니다.
  3. 미납 대시보드 — 메인 화면에서 미납 방 목록과 입금 예정 목록이 표시됩니다.
  4. 월초 리셋 — 매출 현황 카드에서 전체 납부 상태를 일괄 초기화할 수 있습니다. (소유자 전용)
  5. 결제 이력 — 납부 상태 변경 내역이 자동으로 기록되며, CSV로 내보낼 수 있습니다.

6 지급사항 & 옵션

  1. 지급사항 미리값 — 설정 화면에서 이불, 베개, TV 등 자주 사용하는 지급사항을 등록해두면 방 상세에서 빠르게 선택할 수 있습니다.
  2. 옵션 미리값 — 미니외창, 미니내창, 원룸 등 방 타입/옵션도 미리 등록합니다.
  3. 방에 적용 — 방 상세의 지급사항/옵션 영역에서 등록된 항목을 체크하여 적용합니다.

7 팀 협업

  1. 고시원 생성 — 로그인 후 "고시원 만들기"를 선택하여 새 고시원을 생성합니다.
  2. 멤버 초대 — 설정 → 고시원 관리 화면의 초대 코드를 직원에게 공유하세요. 직원이 앱에서 코드를 입력하면 같은 고시원에 합류합니다.
  3. 고시원 전환 — 여러 고시원에 소속된 경우, 메인 화면 좌측 상단의 고시원 이름을 눌러 전환합니다.
  4. 권한 — 소유자는 시설/방 추가, 멤버 관리, 구독 변경, 월초 리셋이 가능합니다. 멤버는 조회 및 수정만 할 수 있습니다.

8 리포트 & 장부

  1. 월별 리포트 — 매출 현황 카드의 "리포트" 버튼으로 월별 마감 리포트를 생성하고 조회합니다.
  2. 수입/지출 장부 — 매출 현황 카드의 "장부" 버튼으로 월별 수입·지출 내역을 관리합니다.
  3. CSV 내보내기 — 방 현황, 결제 이력, 월별 리포트를 CSV 파일로 내보내 공유할 수 있습니다.

9 알림 설정

  1. 납부일 알림 — 설정에서 납부 예정일 며칠 전에 알림을 받을지 설정합니다. (기본 3일 전)
  2. 계약 만료 알림 — 계약 만료일이 다가오면 사전에 알림을 받을 수 있습니다.

? 자주 묻는 질문

  1. Android와 iOS에서 같은 데이터를 사용할 수 있나요? — 네. 같은 Google 계정으로 로그인하면 Android와 iOS 간 모든 데이터가 동기화됩니다.
  2. 방을 실수로 삭제했어요 — 삭제된 방은 복구할 수 없습니다. 삭제 전 확인 대화상자를 꼭 확인해주세요.
  3. 구독을 해지하면 데이터가 사라지나요? — 아닙니다. 데이터는 그대로 유지되며, 무료 플랜 제한을 초과하는 방만 잠금 처리됩니다.
  4. 고시원 소유자를 변경할 수 있나요? — 현재는 소유자 변경이 지원되지 않습니다. 새 고시원을 만들어 이전해주세요.
  5. 구독은 어떻게 관리하나요? — Android는 Google Play 스토어 → 구독, iOS는 설정 → Apple ID → 구독에서 관리할 수 있습니다.

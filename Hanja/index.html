처음부터 끝까지, 막히지 않는 학습 흐름

레벨을 찾고, 매일 복습하고, 손으로 익히고, 모르는 한자는 바로 찾는 — 한자 학습의 모든 단계를 한 앱에서.

01

어디서부터? 먼저 내 레벨을 찾으세요

시작이 가장 막막합니다. 단 5문항 급수 진단으로 지금 내 실력에 딱 맞는 출발 급수를 정해드려요. 너무 쉽거나 어려운 한자에 시간을 낭비하지 않습니다.

  • 8급~1급 중 내 출발 급수 자동 추천
  • 언제든 다시 진단해 목표를 재설정
02

매일, 딱 필요한 만큼만

홈을 열면 '오늘의 학습'이 준비돼 있어요. FSRS 알고리즘이 잊어버릴 때쯤인 복습 한자와 새로 배울 한자를 골라줍니다. 연속 학습 기록으로 습관이 됩니다.

  • 복습 + 신규 한자를 한 세션으로
  • 연속 학습(streak)과 '오늘의 한자'로 동기 부여
03

카드 → 필순 → 퀴즈, 3단계로 확실하게

오늘의 학습은 카드로 익히고, 직접 써보며 손에 새기고, 퀴즈로 확인하는 한 흐름입니다. 눈으로만 스치는 학습이 아니라 기억에 남는 학습이에요.

  • 카드 학습 → 필순 쓰기 → 미니 퀴즈 한 세션
  • 음·훈, 유사자, 부수, 획수 등 다양한 퀴즈 유형
04

손으로 익히는 정확한 필순

획 하나하나를 직접 따라 쓰면 실시간으로 채점해줍니다. 단계별 획순 가이드를 따라가며 올바른 순서를 몸으로 익혀요. Apple Pencil도 지원합니다.

  • KanjiVG 기반 단계별 획순 가이드
  • 실시간 채점 + 틀린 획만 다시 쓰기
05

9,000자, 급수별로 한눈에

8급부터 1급까지 9,000자 이상을 급수별로 정리했어요. 한 글자마다 정확한 대표훈음은 물론, 여러 갈래의 음·뜻(다의어)까지 꼼꼼히 담았습니다.

  • 급수순 · 획수순 · 음순 정렬, 음/훈 검색
  • 대표훈음 + 다의어(여러 음·뜻) 표시
  • 자주 보는 한자는 즐겨찾기로 단어장처럼
06

음을 몰라도, 모양으로 찾기

읽는 법을 모르는 한자도 괜찮아요. 손으로 그리거나 사진으로 찍으면 비슷한 후보를 찾아줍니다. 책에서 마주친 낯선 한자도 바로 검색돼요.

  • 필기 인식 — 획을 따라 쓰면 후보 자동 추천
  • 카메라 인식 — 사진 속 한자도 바로 검색
07

내 진도, 한눈에

얼마나 배웠고 무엇이 남았는지 MY 탭에서 확인하세요. 마스터·학습 중·신규 현황부터 주간 학습량, 급수별 진도까지 한 화면에 담았습니다.

  • 마스터 / 학습 중 / 신규 집계
  • 주간 학습량과 급수별 진도율
